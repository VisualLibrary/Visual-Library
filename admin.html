How to Add a Wallpaper

Step 1 — Upload to Cloudinary

Open Cloudinary → Media Library → go to the correct category folder → upload the wallpaper image. After uploading, click the image and copy its Public ID (shown in the asset details panel).

Step 2 — Edit wallpapers.json on GitHub

Go to your GitHub repo → click wallpapers.json → click the pencil icon to edit in browser. Add a new entry inside the wallpapers array. Copy the format below exactly:

{
  "id": "unique-slug-here",
  "title": "Wallpaper Title Here",
  "category": "marvel",
  "subcategory": null,
  "tags": ["tag one", "tag two", "tag three"],
  "cloudinary_id": "Your_Cloudinary_Public_ID",
  "resolution": "4K",
  "featured": false
}

Make sure to add a comma after the previous entry's closing } before adding your new one.

Step 3 — Save & Deploy

Scroll down on GitHub → click Commit changes → Commit directly to main → click Commit changes. Netlify detects the change and redeploys automatically within 30 seconds.
