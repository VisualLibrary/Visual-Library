How to Add a Wallpaper

Step 1 — Upload to Cloudinary

Go to your Cloudinary Media Library. Upload the wallpaper image into the correct folder (e.g. marvel/). After uploading, click the image and copy its Public ID.

Step 2 — Edit wallpapers.json on GitHub

Go to your GitHub repo → click wallpapers.json → click the pencil icon to edit. Add a new entry to the wallpapers array:

{
  "id": "unique-id-here",
  "title": "Wallpaper Title",
  "category": "marvel",
  "subcategory": null,
  "tags": ["tag1", "tag2"],
  "cloudinary_id": "Your_Public_ID_Here",
  "resolution": "4K",
  "featured": false
}

Step 3 — Commit

Scroll down on GitHub and click Commit changes. The site updates automatically within 30 seconds.
